# Hardware Bulk Change - Quick Start Guide

## What It Does

Replace or remove hardware items (hinges, slides, pulls, handles) across multiple cabinets in seconds instead of minutes.

## Quick Access

1. **Project-Wide**: Click "Change Materials" button in project toolbar
2. **Area-Specific**: Click refresh icon (⟳) on area header

## Basic Steps

### Replace Hardware

1. Select **Scope** (Project / Area / Selected Areas)
2. Choose **"Hardware"** as Material Type
3. Select **Current Hardware** (e.g., "Standard Hinge")
4. Select **New Hardware** (e.g., "Soft-Close Hinge")
5. Click **Preview Changes**
6. Review cost impact
7. Click **Apply Changes**

### Remove Hardware

1. Select **Scope** (Project / Area / Selected Areas)
2. Choose **"Hardware"** as Material Type
3. Select **Current Hardware** (e.g., "Cabinet Pull")
4. Click **"Remove Instead"** button
5. Click **Preview Changes**
6. Confirm removal
7. Click **Apply Changes**

## Key Features

- **Time Savings**: 50+ cabinets in <1 minute
- **Safe**: Only compatible hardware can replace each other
- **Preserved Quantities**: Original quantities never change
- **Full Preview**: See cost impact before applying
- **Flexible Scope**: Entire project, one area, or multiple areas

## Hardware Categories

- **Hinges** → Can only replace with other hinges
- **Slides** → Can only replace with other slides
- **Pulls** → Can only replace with other pulls
- **Handles** → Can only replace with other handles

## Common Uses

1. **Upgrade**: Standard slides → Soft-close slides
2. **Downgrade**: Premium hinges → Standard hinges
3. **Remove**: Delete pulls when client provides their own
4. **Standardize**: Replace mixed hardware with consistent options

## Safety Notes

- ⚠️ Changes over 20% trigger confirmation
- ⚠️ Removals require additional confirmation
- ⚠️ Always preview before applying
- ✓ All changes are logged for audit
